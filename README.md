# mangapy

Manga downloader

- fanfox.net
- mangapark.net

## Installation

Install from local folder

```
pip install .
```

Install from local folder (editable mode)

```
pip install -e .
```

Install a local setup.py into your virtual environment/Pipfile:

```
pipenv install -e .
```

Run tests:

```
pipenv run pytest
```

## Usage

```
mangapy bleach -c 0-1 -o "~/Downloads"
```
